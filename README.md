# Material de Apoyo para Investigación

Repositorio centralizado con plantillas, metodologías y recursos para proyectos de investigación.

## Estructura

- `plantillas/` - Documentos reutilizables para proyectos
- `metodologias/` - Protocolos y métodos de investigación
- `herramientas/` - Guías de software y plataformas
- `buenas-practicas/` - Estándares y mejores prácticas
- `investigacion-reproducible/` - Métodos para investigación reproducible
- `recursos-aprendizaje/` - Material educativo y referencias
- `material-a-revisar/` - Contenido en proceso de organización
