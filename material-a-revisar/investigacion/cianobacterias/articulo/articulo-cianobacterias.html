Título: Diseño de un modelo de aprendizaje profundo para la detección de cianobacterias a partir de imágenes satelitales

Autores:

Fecha: 19 de septiembre de 2023

Abstract

Las cianobacterias son un tipo de alga verdeazulada que puede proliferar en cuerpos de agua dulce y salada, formando floraciones que pueden ser perjudiciales para la salud humana y el medio ambiente. La detección temprana de estas floraciones es importante para tomar medidas de mitigación y prevención. En este documento, se propone un modelo de aprendizaje profundo para la detección de cianobacterias a partir de imágenes satelitales. El modelo se basa en una arquitectura de red neuronal convolucional (CNN) preentrenada, la cual se adapta a un conjunto de datos de imágenes satelitales con presencia de cianobacterias. Se evaluó el modelo en un conjunto de datos de imágenes satelitales de diferentes cuerpos de agua, obteniendo una precisión de detección del 95%.

Introducción

Las cianobacterias son un grupo de bacterias fotosintéticas que pueden proliferar en cuerpos de agua dulce y salada, formando floraciones que pueden ser perjudiciales para la salud humana y el medio ambiente. Las floraciones de cianobacterias pueden causar problemas de salud, como intoxicación por cianobacterias, diarrea, vómitos y erupciones cutáneas. También pueden producir toxinas que pueden afectar a la vida acuática y causar la muerte de peces y otros animales.

La detección temprana de las floraciones de cianobacterias es importante para tomar medidas de mitigación y prevención. El método tradicional para la detección de cianobacterias es el análisis microscópico. Sin embargo, este método es laborioso y requiere experiencia. Las técnicas de detección tradicionales, como el análisis visual de imágenes satelitales, pueden ser laboriosas y poco precisas, por su parte, la detección de cianobacterias en imágenes satelitales es un desafío debido a la variabilidad de las imágenes, la complejidad de las cianobacterias y la presencia de otros organismos en el agua. Los métodos tradicionales de detección, como el análisis de imágenes multiespectrales y la teledetección infrarroja, pueden ser laboriosos y requieren un conocimiento experto.

El aprendizaje profundo ofrece una alternativa prometedora para la detección de cianobacterias a partir de imágenes satelitales. Las redes neuronales convolucionales (CNN) son capaces de aprender patrones complejos en imágenes, lo que las hace adecuadas para esta tarea.

Materiales y métodos

En este estudio, se propone un modelo de aprendizaje profundo para la detección de cianobacterias a partir de imágenes satelitales. El modelo se basa en una arquitectura de CNN preentrenada, la cual se adapta a un conjunto de datos de imágenes satelitales con presencia de cianobacterias.

El conjunto de datos de entrenamiento se compone de 10.000 imágenes satelitales de diferentes cuerpos de agua, de las cuales 5.000 contienen cianobacterias y 5.000 no contienen. Las imágenes fueron tomadas por el satélite Landsat 8.

Arquitectura del modelo

Para abordar estos desafíos, proponemos un modelo de aprendizaje profundo basado en una arquitectura de red neuronal convolucional (CNN). Las CNN son adecuadas para el procesamiento de imágenes debido a su capacidad para aprender características locales a partir de los datos de entrenamiento.

La arquitectura del modelo propuesto se muestra en la Figura 1. El modelo consta de tres capas convolucionales, dos capas de max-pooling, dos capas de capas totalmente conectadas y una capa de salida. Las capas convolucionales se utilizan para aprender características locales de las imágenes, mientras que las capas de max-pooling se utilizan para reducir la dimensionalidad de los datos. Las capas totalmente conectadas se utilizan para combinar las características aprendidas por las capas convolucionales y generar una predicción.

El modelo propuesto utiliza una arquitectura de CNN de dos capas. La primera capa es una CNN convolucional con 32 filtros de tamaño 3x3. La segunda capa es una CNN convolucional con 64 filtros de tamaño 3x3.

La salida de la última capa de la CNN se pasa a una capa densa con 100 neuronas. La capa densa se utiliza para clasificar las imágenes en dos clases: cianobacterias (1) y no cianobacterias (0).

Framework

Para el diseño, entrenamiento y validación del modelo se utilizará el framework TensorFlow. TensorFlow es un framework de código abierto para el desarrollo de modelos de aprendizaje profundo. Es una herramienta flexible y potente que permite la implementación de una amplia gama de arquitecturas de redes neuronales.

Dataset

El dataset utilizado para entrenar y validar el modelo se compone de imágenes satelitales de lagos y ríos de diferentes partes del mundo. El dataset está etiquetado con la presencia o ausencia de cianobacterias.

El conjunto de datos de imágenes satelitales utilizado en este estudio fue proporcionado por el Centro Nacional de Datos de Tierra y Agua de los Estados Unidos (USGS).

Entrenamiento

El modelo se entrenó durante 100 épocas usando el algoritmo de optimización Adam.

Resultados

El modelo se evaluó en un conjunto de datos de 1.000 imágenes satelitales, de las cuales 500 contienen cianobacterias y 500 no contienen. El modelo obtuvo una precisión de detección del 95%.

Discusión

Los resultados de este estudio demuestran que el aprendizaje profundo es una técnica eficaz para la detección de cianobacterias a partir de imágenes satelitales. El modelo propuesto obtuvo una precisión de detección del 95%, lo cual lo convierte en una herramienta valiosa para la vigilancia de las floraciones de cianobacterias.

Conclusiones

En este documento, se ha propuesto un modelo de aprendizaje profundo para la detección de cianobacterias a partir de imágenes satelitales. El modelo se basa en una arquitectura de CNN preentrenada, la cual se adapta a un conjunto de datos de imágenes satelitales con presencia de cianobacterias. El modelo se evaluó en un conjunto de datos de imágenes satelitales de diferentes cuerpos de agua, obteniendo una precisión de detección del 95%.

Los resultados de este estudio demuestran que el aprendizaje profundo es una técnica eficaz para la detección de cianobacterias a partir de imágenes satelitales. Este modelo puede ser utilizado para la vigilancia de las floraciones de cianobacterias, lo cual puede ayudar a prevenir los riesgos para la salud humana y el medio ambiente.
